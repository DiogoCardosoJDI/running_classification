# running_classification

PASSO 1:
Certifique-se que tenho o python 3.11 ou superior instalado;
PASSO 2:
Dê um clique duplo no arquivo instalar.bat para que o ambiente seja preparado e as dependencias sejam instaladas;
PASSO 3:
Certifique-se que o arquivo planilha_modelo.xlsx atualizada se encontra na pasta data_charge;
PASSO 4:
Dê um clique duplo no arquivo iniciar.bat para que seja iniciada a execução do programa;


Após iniciado o programa:
Antes de clicar no botão CARREGAR DADOS selecione a opção "Incluir Advogados na Clasificação Geral?" caso os mesmos participem em conjunto com os demais participantes e não haja classificação segregada para advogados;
Clique no botão CARREGAR DADOS para gerar o banco de dados atualizado da prova;
Ao dar a largada, clique no botão INICIAR PROVA, irá aparecer um popup mostrando o horário de inicio da prova, basta clicar em "ok" para fechar;
O tempo da prova irá ficar aparecendo enquanto não for finalizado a corrida;
No campo "Número do Participante" digite o númeo do corredor e aperte "ENTER" ou clique em REGISTRAR CHEGADA assim que o mesmo finalizar a prova;
Ao termino do evento, clique em FINALIZAR & GERAR RELATÓRIOS para que seja finalizado o evnto e os relatórios de classificação sejam gerados.

Após geração dos relatórios os mesmos estarão na pasta raiz da aplicação c:\corrida\running_classification, os arquivos gerados estão no formato .PDF;
